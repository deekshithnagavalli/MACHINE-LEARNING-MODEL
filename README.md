# MACHINE-LEARNING-MODEL

DATASET : "NYPD COMPLAINT DATA CURRENT"

I took the "NYPD COMPLAINT DATA CURRENT" dataset. It contains more than 30 columns and it is available in CSV format. 
The dataset includes information such as crime time, location, offense type, crime severity, jurisdiction details, and demographic details of suspects and victims.
I had gone through the entire columns of this dataset and understood what will be the problem statement.

 Problem Statement :
Build a predictive model to estimate the type and likelihood of crime occurring at a given location and time using historical crime data. The model will analyze patterns in past records to understand when and where crimes are more likely to happen.
This project aims to use factors such as time, location, and crime-related details to make accurate predictions. By identifying these patterns, the system can help in early awareness and better planning.

 Title :
CRIME OCCURANCE PREDICTION

 Goal :
The main goal of this project is to predict when and where crimes are likely to occur using past data and try to reduce the crime. This model will help in understanding high-risk areas and peak crime times, making it easier to take preventive measures. It focuses on using data to provide meaningful insights rather than just raw information.

The project also aims to support better decision-making for authorities by highlighting crime trends and possible future occurrences. In addition, it can help the public stay aware of unsafe areas and times, promoting safer environments.


 Columns Impacting the Project :
 
* Time: CMPLNT_FR_DT, CMPLNT_FR_TM
* Location: BORO_NM, LATITUDE, LONGITUDE
* Crime Details: OFNS_DESC, LAW_CAT_CD
* Demographics: SUSP_AGE_GROUP, SUSP_SEX, VIC_AGE_GROUP


Project Helpful For:

 * Helps police identify high crime areas and focus on those locations for better monitoring
 * Assists authorities in planning and improving safety measures in risky zones
 * Supports better allocation of resources like patrol teams and surveillance 
 * Helps in identifying peak crime times so preventive actions can be taken in advance
 * Increases public awareness about crime-prone areas and unsafe timings


 Conclusion:
 
This project helps in understanding crime patterns and can be useful for improving public safety.It also enables data-driven decision-making by identifying hidden patterns and trends in crime data.
